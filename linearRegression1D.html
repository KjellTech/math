Linear Regression 1D

\[y_i = a + bx_i \] Square Error: \[SE = \Sigma (y_i - a - bx_i)^2 \] Minimizing by requiring that derivatives with respect to to a and b are 0: \[\Sigma 2(y_i - a - bx_i)(-1) = 0 \] \[\Sigma 2(y_i - a - bx_i)(-x_i) = 0 \] \[ a = \bar{y} - b\bar{x}\] where the bar denotes average \[ \bar{xy} - a\bar{x} - b\bar{xx} = 0\] \[ \bar{xy} - (\bar{y} - b\bar{x})\bar{x} - b\bar{xx} = 0 \] \[ b=(\bar{xy} - \bar{x}\bar{y}) / (\bar{xx} - \bar{x}\bar{x})\]
